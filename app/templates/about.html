About

This project uses basic natural language processing to generate semi-plausible lyrics for a given artist.

How it works

Enter an artist. The algorithm scrapes the top 10 most popular songs for that artist using the RapGenius API. Then, the algorithm divides the input lyrics into k-grams, and for each k-gram, forms a frequency table for the following character.

When generating new lyrics, the algorithm looks at the current k-gram and selects a random character based off the weighted distribution of the frequency table.

The algorithm incorporates the character into the next k-gram and repeats the process.

Called a Markov chain, the algorithm produces new lyrics.

Project Design

I coded the Markov chain in Python and hosted it on Heroku with the Flask micro web framework. I wrote a CSS/HTML/JavaScript GUI to mimic the effect of the Terminal when originally testing the algorithm. For the animation, I used the Typed.js library, and for the RapGenius API, I used an existing scraper.
